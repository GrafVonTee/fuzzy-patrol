# FuzzyPatrol

Agents with AI trying to minimize both Danger Level and their own Thurst Level

Controls:
- Esc -- Quit Game
- LMB on the right area -- Spawn Visible Enemy
- RMB on the right area -- Spawn temporary Invisible Enemy
